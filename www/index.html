Give a second life to your light event wristband!

If you have light even wristband such as a PixMob you can easily use it at home with this web.

You will need:

  1. An Arduino-compatible microcontroller.
  2. A 940nm IR emitter for the Arduino. You could set up a raw IR led or use a board like the transmitter piece of this.

Instruction to use

  1. Upload this sketch to your Arduino compatible board (connected to an infrarred sensor). If you are using an ESP32 board, then use this sketch.
  2. Connect the board to this webpage using the [Connect board] button in the main screen.
  3. Set the main effect and tail code you desire and press Send!
